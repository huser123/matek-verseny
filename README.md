# 🪢 MatekVerseny – Kötélhúzás

Kiszolálós, browsőrben futó, kétcsapatos matematikajáték alsó tagozatos gyerekeknek.

## 🎮 Játékleírás

Két csapat egymással szemben old meg egyszerű matematikai feladatokat. Minden helyes válasz előnyt ad a csapatnak – a középen látható kötélhúzás animáció mutatja, melyik csapat húzza el a másikat.

## ⚙️ Beállítások

- **Műveletek:** összeadás, kivonás, szorzás, osztás
- **Nehézség:** könnyű, közepes, nehéz
- **Csapatnevek** szerkeszthetők
- **Időkorlát** és **győzelmi ponthatár** szabad
onfigurálható

## 🚀 Futtatás
